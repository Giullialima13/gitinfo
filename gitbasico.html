Introdução ao Git

Aprenda o básico de Git.

O que é Git?

Git é um sistema de controlo de versões distribuído que permite guardar, acompanhar e gerenciar alterações no código de forma eficiente. É usado por programadores, equipas de desenvolvimento e projetos open-source no mundo inteiro.

Conceitos Fundamentais

1. Repositório (Repository)

É o local onde os arquivos do projeto e o seu histórico são armazenados.

2. Commit

Um commit representa uma alteração registrada no histórico. É como tirar uma “fotografia” do estado do projeto naquele momento.

3. Branch (Ramo)

Branches permitem criar versões paralelas do projeto. São úteis para desenvolver funcionalidades sem interferir no código principal.

4. Merge

Merge é o processo de unir alterações de um branch para outro.

5. Clone & Pull

  • Clone: copiar um repositório para o teu computador.
  • Pull: atualizar o teu repositório local com as últimas mudanças do repositório remoto.

Comandos Básicos de Git

Inicializar um repositório:

git init

Clonar um repositório remoto:

git clone URL_DO_REPOSITORIO

Adicionar alterações:

git add.

Criar um commit:

git commit -m "mensagem do commit"

Ver o status:

git status

Criar um branch:

git branch nome-do-branch

Trocar de branch:

git checkout nome-do-branch

Fazer merge:

git merge nome-do-branch
